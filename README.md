Aplicación móvil desarrollada con React Native y Expo, creada como práctica académica para aplicar conceptos de desarrollo móvil.

 Cómo probar la aplicación
Instalar Expo Go en tu celular.
Escanear el código QR que aparece en la terminal o navegador.
También puedes usar un emulador Android si lo tienes configurado.

Objetivo del proyecto
El objetivo de esta aplicación es reforzar conocimientos en:
Navegación entre pantallas
Componentes funcionales
Organización modular del proyecto
Uso de estilos en React Native
